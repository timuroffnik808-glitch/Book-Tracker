# Book Tracker

**Автор:** [Абдуллаев Темурбек]

### Описание
Приложение для учета прочитанных книг. Позволяет сохранять названия, авторов, жанры и объем книг, а также фильтровать список для быстрого поиска.

### Инструкция по запуску
1. Установите Python (версия 3.x).
2. Скачайте файл `book_tracker.py`.
3. Запустите приложение командой:
   ```bash
   python book_tracker.py
   ```

### Функционал
- **Добавление:** Валидация пустых полей и числового формата страниц.
- **Хранение:** Автоматическое сохранение в `books.json`.
- **Фильтрация:** Поиск по жанру и минимальному количеству страниц.
